API 使用說明

API 請求格式：http://{域名}/{服務名稱}/{令牌值}/{路徑}

示例：如果要訪問 example-api 服務的 /users 路徑，令牌值為 abcdef123456，則 URL 為：

http://your-domain.com/example-api/abcdef123456/users

提示：令牌會在到期日期後失效，請確保在到期前更新令牌。
